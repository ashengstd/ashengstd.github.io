常用软件和工具包

发表于2022-11-19|更新于2022-11-19
|阅读量:

本文作为博客的第一篇文章, 主要介绍我所用到的一些软件(包括在Win11和Linux下的), 以便未来重装和分享经验.

Windows

  • 基础应用

    • Mircosoft Edge / Chrome

      基于Chromium内核的两款浏览器, Edge更适合没有代理的情况使用, 同步方便也全平台, 并且拓展商店访问也正常, 不过感觉设计没Chrome那么好看, 建议有代理有Chrome.

      这里给出油猴的官网, 以及一个国内的拓展商店.

      Tampermonkey • 首页

      Crx搜搜 - 一个牛X的扩展和应用商店

    • Xdown

      文件下载器, 支持BT下载和直链, 配合油猴插件可以实现百度网盘加速

      不过效率好像并不是最好的一类, 也可以试试IDM, Bitcomet, Qbittorrent Enhanced.

    • Watt Toolkit - 瓦特工具箱(Steam++官网)

      原Steam++, 功能不至加速Steam, 还可以加速Github和Microsoft Store.

    • Clash

      代理工具.

      Dreamacro/clash: A rule-based tunnel in Go.

      Windows UI版本:

      Dreamacro/clash: A rule-based tunnel in Go.

  • 视频音乐

    • VLC Media Player

      开源视频播放器, 目前还没摸索出来太多优势, 界面偏简陋不过用起来很轻, 也是全平台.

    • Cider

      Apple Music的第三方版本, 相比难用的iTunes真的是一股清流, Microsoft Store可以下载, 不过是收费的, 也可以去Github去下载Release版本, 免费.

      ciderapp/Cider: A new cross-platform Apple Music experience based on Electron and Vue.js written from scratch with performance in mind. 🚀

    • OBS Studio

      最强屏幕录制和推流工具,还是开源的. 强推!

  • 游戏娱乐
    • Steam
    • Xbox
    • Epic
    • EA
  • 编程开发

    • Visual Studio Code - Code Editing. Redefined

      文本编辑器但不止编辑, 配合Mingw-w64,Code Runner可以实现C语言的编译和运行

      以下为高级用途:

      • 首先就是Mingw-w64

      众所周知, Linux的C语言编译器GCC 很好用, 而Mingw-w64就是GCC的Windows版本, 即Minimalist GNU for Windows

      MinGW-w64

      下载:

      Online Installer

      压缩文件: Releases · niXman/mingw-builds-binaries (github.com)

      压缩文件自己解压放到想要的地方, 编辑系统环境变量添加bin到PATH里即可.

      安装或者下载时可能会遇到几个选项, Version即版本, Architecture即系统架构版本, 不懂就选x86_64, Threads即操作系统的接口协议, 不懂就选Win32, Exception即异常处理模型, 建议seh, Build Version默认

      • Visual Studio 2022 IDE - 适用于软件开发人员的编程工具 (microsoft.com)

      被称为宇宙第一IDE, 不过我感觉Clion更好看就是了.

      官网选择Community版本, 免费, 如果你财力丰厚也可以选择Professional或者Enterprise版本.

      选择组件的时候选择使用C++ 的桌面开发, 后续有图会放上来

      • Jetbrains 全家桶

      Jetbrains家的软件, 就一个字, 优雅.IDEA也是最强JAVA IDE.

      这里贴上官网JetBrains: 软件开发者和团队的必备工具

      以及Toolbox, 相当于Jetbrains系软件的管理工具,强推

      JetBrains Toolbox App：轻松管理您的工具

      Jetbrains还有学生认证,有教育邮箱的话可以免费用.

Linux (Debian)

本文Linux为WSL-Debian发行版,为Microsoft定制内核,与GNU / Linux稍有不同,同时也命令也仅在Debian系下可用.

1. WSL-Debian安装

若要更新到 WSL 2，需要运行 Windows 10。
对于 x64 系统:版本 1903 或更高版本,采用 内部版本 18362 或更高版本.
对于 ARM64 系统:版本 2004 或更高版本,采用 内部版本 19041 或更高版本。

低于 18362 的版本不支持 WSL 2. 使用 Windows Update 助手更新 Windows 版本.

  • 启用适用于 Linux 的 Windows 子系统
    以管理员身份打开 PowerShell 并运行：
  • 启用虚拟机功能
  • 下载 Linux 内核更新包

x86架构 WSL2 Linux kernel update package for x64 machines
如果是ARM架构,则下载 ARM架构更新包

  • 设置WSL2为默认WSL版本
  • 在Microsoft Store 安装想要的Linux发行版,如Debian.

2. 软件包

  • GCC / GNU C Compile
    LInux下经典C/C++编译器.
  • Make/Cmake
    编译项目和很多包的前置依赖,也可以用来处理项目编译.
